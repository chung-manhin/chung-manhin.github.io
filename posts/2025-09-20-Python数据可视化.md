# Python 数据可视化：Matplotlib 与 Seaborn 实战

数据可视化是理解数据的第一步。这篇文章整理了使用 Python 两大可视化库的常用方法。

## Matplotlib 基础

Matplotlib 是 Python 最经典的绘图库，功能强大但语法较底层。

### 基本折线图

```python
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)

plt.figure(figsize=(10, 6))
plt.plot(x, y, color='#2A7F6F', linewidth=2, label='sin(x)')
plt.fill_between(x, y, alpha=0.1, color='#2A7F6F')
plt.title('正弦函数', fontsize=16)
plt.xlabel('x')
plt.ylabel('y')
plt.legend()
plt.grid(True, alpha=0.3)
plt.show()
```

### 子图排列

```python
fig, axes = plt.subplots(2, 2, figsize=(12, 10))
# 在每个子图中绘制不同的图表
```

## Seaborn 进阶

Seaborn 基于 Matplotlib，提供更高级的统计图表和更美观的默认样式。

### 热力图

```python
import seaborn as sns

data = np.random.rand(8, 8)
sns.heatmap(data, annot=True, cmap='YlGnBu', fmt='.2f')
plt.title('相关性热力图')
```

### 分布图

```python
sns.histplot(data=df, x='value', hue='category', kde=True)
```

## 美化技巧

1. **选择合适的配色**：使用 `cmap` 参数选择色彩方案
2. **调整字体大小**：通过 `rcParams` 全局设置
3. **添加注释**：使用 `plt.annotate()` 标注关键点
4. **保存高清图片**：`plt.savefig('plot.png', dpi=300, bbox_inches='tight')`

> 好的可视化不是越花哨越好，而是要清晰地传达数据背后的信息。

## 总结

Matplotlib 适合精细控制，Seaborn 适合快速出图。实际项目中两者经常搭配使用。
