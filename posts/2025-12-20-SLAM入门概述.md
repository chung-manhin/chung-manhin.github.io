# SLAM 入门概述：从原理到开源方案选型

SLAM（Simultaneous Localization and Mapping，同时定位与建图）是移动机器人和自动驾驶的核心技术之一。本文梳理 SLAM 的基本原理和主流开源方案。

## SLAM 要解决什么问题

机器人在未知环境中移动时，需要同时回答两个问题：

1. **我在哪？**（定位）
2. **周围是什么？**（建图）

这两个问题相互依赖：准确定位需要地图，而建图又需要准确的位姿。SLAM 就是同时解决这个"鸡生蛋"问题的算法框架。

## SLAM 系统架构

一个典型的 SLAM 系统包含：

### 前端（数据关联）

- **特征提取**：从传感器数据中提取有用信息
- **数据匹配**：帧间匹配、回环检测
- **运动估计**：计算相邻帧之间的位姿变化

### 后端（优化）

- **位姿图优化**：用图优化（g2o、GTSAM）全局优化所有位姿
- **回环校正**：检测到回环后修正累积漂移

### 建图

- **稀疏地图**：特征点地图，用于定位
- **稠密地图**：点云/体素地图，用于导航和避障
- **语义地图**：带有语义标签的地图

## 传感器分类

| 传感器 | 优点 | 缺点 |
|--------|------|------|
| 单目相机 | 便宜、轻量 | 无尺度信息、初始化困难 |
| 双目相机 | 有深度信息 | 计算量大、基线限制 |
| RGB-D | 直接获取深度 | 室外效果差 |
| 激光雷达 | 精度高、距离远 | 贵、数据量大 |
| IMU | 高频、不受环境影响 | 漂移严重 |

实际应用中通常多传感器融合。

## 主流开源方案

### ORB-SLAM3

- **类型**：视觉 SLAM
- **传感器**：单目/双目/RGB-D + IMU
- **特点**：特征点法，精度高，支持多地图
- **适用**：室内外通用，学术研究首选
- **论文**：Campos et al., 2021

### Cartographer

- **类型**：激光 SLAM
- **传感器**：2D/3D 激光雷达 + IMU
- **特点**：Google 出品，子图+回环优化，工程成熟
- **适用**：室内机器人导航，ROS 集成好

### LIO-SAM

- **类型**：激光惯性里程计
- **传感器**：3D 激光 + IMU + GPS（可选）
- **特点**：紧耦合，因子图优化，实时性好
- **适用**：室外场景，自动驾驶

### VINS-Mono / VINS-Fusion

- **类型**：视觉惯性 SLAM
- **传感器**：单目/双目 + IMU
- **特点**：紧耦合 VIO，滑动窗口优化
- **适用**：无人机、AR/VR

## 方案选型建议

```
室内导航 + 2D 雷达 → Cartographer
室内导航 + 相机    → ORB-SLAM3
室外建图 + 3D 雷达 → LIO-SAM
无人机/AR         → VINS-Fusion
学习研究          → ORB-SLAM3（代码规范，论文详细）
```

## 学习路线

1. **数学基础**：线性代数、矩阵论、李群李代数
2. **入门教材**：《视觉SLAM十四讲》（高翔）
3. **动手实践**：跑 ORB-SLAM3 + EuRoC 数据集
4. **深入理解**：图优化、边缘化、IMU 预积分
5. **工程落地**：ROS2 + Cartographer/Nav2

## 小结

SLAM 是一个涉及几何、优化、概率论的综合性课题。建议从视觉 SLAM 入手，先理解核心原理，再根据实际需求选择合适的传感器和方案。
