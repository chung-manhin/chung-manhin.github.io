## 遇到的问题

## 🎯 目标

把本地 Hexo 博客部署到 GitHub Pages，让全世界都能访问。

## 🕳️ 实际踩的坑

### 1. 404 并不是配置错了

- 症状：部署完成后访问 `https://<用户名>.github.io` 直接 404。
- 原因：**GitHub Pages 启用后需要 1~2 分钟生效**。
- 解决：冲杯咖啡再刷新，页面神奇出现。

### 2. Git 下载后依然提示 "不是内部或外部命令"

- 症状：PowerShell 里执行 `git --version` 报错。
- 原因：**没有把 Git 的 `bin` 和 `cmd` 目录加到系统 Path**。
- 解决：
  1. 把
     `C:\Program Files\Git\bin`
     `C:\Program Files\Git\cmd`
     追加到系统环境变量 → Path。
  2. **重启终端**（关键一步，否则新变量不生效）。

### 3. `hexo deploy` 报错 "Author identity unknown"

- 症状：Git 不知道我是谁。
- 原因：**Git 没有配置用户信息**。
- 解决：执行

```bash
git config --global user.name "你的用户名"
git config --global user.email "你的github注册邮箱"
```
